# Практические занятия — Технологии виртуализации

## Как пользоваться этой папкой

Каждая папка `Пара_N` — это один практический урок (80 минут).
Группы идут последовательно: где бы ни начала группа — начинает с Пары 1.

| Пара | Тема | Время |
|------|------|-------|
| Пара 1 | Linux-основы контейнеризации (namespaces, cgroups, chroot) | 80 мин |
| Пара 2 | Docker: образы, Dockerfile, запуск контейнеров | 80 мин |
| Пара 3 | Docker: сети, volumes, docker-compose | 80 мин |
| Пара 4 | Kubernetes: установка кластера, первые поды | 80 мин |
| Пара 5 | Kubernetes: Deployment, Service, Ingress | 80 мин |
| Пара 6 | Kubernetes: ConfigMap, Secret, PV/PVC | 80 мин |
| Пара 7 | Безопасность: RBAC, NetworkPolicy, Falco | 80 мин |
